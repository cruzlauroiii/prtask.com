Come funziona

Getting Started

1

Esplora le attività aperte

I clienti pubblicano attività di sviluppo con ricompense. Sfoglia per complessità, linguaggio o importo della ricompensa.

2

Invia una Pull Request

Scrivi il tuo codice e invia una pull request pubblica. Più sviluppatori possono inviare PR per la stessa attività.

3

Punteggio di complessità

La tua PR viene valutata automaticamente in base ai file modificati, alle righe aggiunte, alla difficoltà del linguaggio e alla copertura dei test.

4

Il cliente esamina e approva

Il cliente esamina tutte le PR inviate e sceglie la migliore da approvare. All'approvazione, al cliente viene addebitato automaticamente l'importo della ricompensa.

5

Ricevi il pagamento

Lo sviluppatore riceve il pagamento tramite il metodo preferito. Sblocca badge di successo e scala la classifica dei guadagni con ogni PR approvata.

Per i clienti

  1. Accedi con GitHub o Google e configura il tuo metodo di pagamento
  2. Crea un'attivita con descrizione, link al repository e importo della ricompensa
  3. Gli sviluppatori inviano pull request pubbliche per la tua attivita
  4. Esamina tutte le pull request inviate e approva la migliore
  5. Ti viene addebitata automaticamente la ricompensa e lo sviluppatore viene pagato
