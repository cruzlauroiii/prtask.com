계정을 연결하여 수익을 얻거나 작업을 게시하세요.

Google로 로그인 GitHub으로 로그인 Microsoft로 로그인 Okta로 로그인

개발자용

작업을 둘러보고, 맡고, PR을 제출하고 보수를 받으세요.

클라이언트용

보상이 있는 작업을 게시하고 양질의 코드를 받으세요.

왜 PrTask인가?

PrTask는 개발자를 유료 오픈소스 및 프라이빗 작업과 연결합니다. 코드를 제출하고, 수입을 얻고, 포트폴리오를 구축하세요.

수입 얻기

병합된 풀 리퀘스트마다 보상을 받습니다. 보상은 사전에 자금이 마련되고 병합 시 해제됩니다.

플랫폼 수수료 없음

개발자는 보상의 100%를 보유합니다. PrTask는 클라이언트에게 청구하며 기여자에게는 청구하지 않습니다.

공개 PR

모든 작업은 GitHub의 실제 풀 리퀘스트로 전달됩니다. 기여는 가시적이고 검증 가능합니다.

작동 방식

1

로그인

선호하는 SSO 제공업체로 인증하여 PrTask 계정을 만듭니다.

2

작업 탐색

언어, 프레임워크 또는 보상 금액으로 필터링하여 사용 가능한 작업을 탐색합니다.

3

PR 제출

작업을 클레임하고 코드를 작성하여 대상 저장소에 풀 리퀘스트를 제출합니다.

4

보상 받기

PR이 검토되고 병합되면 보상이 자동으로 계정에 해제됩니다.

보안 및 신뢰

PrTask는 OAuth 기반 SSO만 사용합니다. 비밀번호를 저장하지 않습니다. 모든 결제는 작업이 검증될 때까지 에스크로에 보관됩니다.
