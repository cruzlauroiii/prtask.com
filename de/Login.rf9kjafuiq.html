Warum PrTask?

PrTask verbindet Entwickler mit bezahlten Open-Source- und privaten Aufgaben. Code liefern, Geld verdienen und Ihr Portfolio aufbauen.

Geld verdienen

Werden Sie fuer jeden zusammengefuehrten Pull Request bezahlt. Praemien werden im Voraus finanziert und bei der Zusammenfuehrung freigegeben.

Keine Plattformgebuehren

Entwickler behalten 100% der Praemie. PrTask berechnet Kunden, niemals Beitragende.

Oeffentliche PRs

Alle Arbeit wird als echte Pull Requests auf GitHub geliefert. Ihre Beitraege sind sichtbar und ueberpruefbar.

So funktioniert es

1

Anmelden

Authentifizieren Sie sich mit Ihrem bevorzugten SSO-Anbieter, um Ihr PrTask-Konto zu erstellen.

2

Aufgaben durchsuchen

Erkunden Sie verfuegbare Aufgaben, gefiltert nach Sprache, Framework oder Praemienbetrag.

3

PRs einreichen

Beanspruchen Sie eine Aufgabe, schreiben Sie den Code und oeffnen Sie einen Pull Request gegen das Ziel-Repository.

4

Bezahlt werden

Sobald Ihr PR ueberprueft und zusammengefuehrt wird, wird die Praemie automatisch auf Ihr Konto freigegeben.

Sicherheit und Vertrauen

PrTask verwendet ausschliesslich OAuth-basiertes SSO. Wir speichern niemals Passwoerter. Alle Zahlungen werden treuhaenderisch verwahrt.
