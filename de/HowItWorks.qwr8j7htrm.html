So funktioniert es

Getting Started

1

Offene Aufgaben durchsuchen

Kunden veröffentlichen Entwicklungsaufgaben mit Prämien. Durchsuchen Sie nach Komplexität, Sprache oder Prämienhöhe.

2

Pull Request einreichen

Schreiben Sie Ihren Code und reichen Sie einen öffentlichen Pull Request ein. Mehrere Entwickler können PRs für dieselbe Aufgabe einreichen.

3

Komplexitätsbewertung

Ihr PR wird automatisch bewertet, basierend auf geänderten Dateien, hinzugefügten Zeilen, Sprachschwierigkeit und Testabdeckung.

4

Kundenprüfung und Genehmigung

Der Kunde überprüft alle eingereichten PRs und wählt den besten zur Genehmigung aus. Bei Genehmigung wird dem Kunden automatisch der Prämienbetrag berechnet.

5

Bezahlt werden

Der Entwickler erhält die Zahlung über seine bevorzugte Auszahlungsmethode. Schalten Sie Auszeichnungen frei und steigen Sie mit jedem genehmigten PR in der Verdienstrangliste auf.

Fuer Kunden

  1. Melden Sie sich mit GitHub oder Google an und richten Sie Ihre Zahlungsmethode ein
  2. Erstellen Sie eine Aufgabe mit Beschreibung, Repository-Link und Praemienbetrag
  3. Entwickler reichen oeffentliche Pull Requests fuer Ihre Aufgabe ein
  4. Ueberpruefen Sie alle eingereichten Pull Requests und genehmigen Sie den besten
  5. Ihnen wird automatisch die Praemie berechnet und der Entwickler wird bezahlt
