Ihr PrTask Dashboard - Plattformuebersicht und Aktivitaetsverfolgung

Das Dashboard bietet einen zentralen Ueberblick ueber die Plattformaktivitaet auf PrTask. Auf einen Blick sehen Sie die Gesamtzahl der aktiven Aufgaben, die Anzahl der registrierten Benutzer, den gesamten Praemienpool und die Anzahl der eingereichten Pull Requests.

Fuer Entwickler dient das Dashboard als Ausgangspunkt zur Bewertung der aktuellen Arbeitslage. Fuer Kunden bietet das Dashboard Einblick in den Status erstellter Aufgaben, einschliesslich ausstehender Pull Requests.

Das Dashboard aktualisiert sich dynamisch, wenn neue Aufgaben erstellt, Pull Requests eingereicht und Praemien ausgezahlt werden. Das Dashboard dient weiterhin als Ihre Kommandozentrale fuer alle Aspekte Ihrer PrTask-Aktivitaet.
