Über PrTask

PrTask ist eine Plattform, auf der Entwickler durch das Einreichen von Pull Requests bei echten Projekten Geld verdienen.

Unsere Mission

PrTask existiert, um Softwareentwicklung für alle zugänglich, fair und lohnend zu machen. Wir glauben, dass talentierte Entwickler überall gleichberechtigten Zugang zu sinnvoller Arbeit und transparenter Vergütung verdienen, unabhängig davon, wo sie leben.

Was wir tun

Kunden veröffentlichen Entwicklungsaufgaben mit Prämien. Entwickler beanspruchen Aufgaben, schreiben Code und reichen Pull Requests ein. Sobald ein PR überprüft, genehmigt und zusammengeführt wird, wird der Entwickler basierend auf der Komplexität seiner Arbeit bezahlt.

Unsere Geschichte

PrTask wurde mit einer einfachen Idee gegruendet: Was waere, wenn jeder Codebeitrag eine klare, garantierte Praemie haette? Entstanden aus der Frustration unbezahlter Open-Source-Arbeit, wurde PrTask ins Leben gerufen, um einen Marktplatz zu schaffen, auf dem Pull Requests die universelle Waehrung der Arbeit sind.

Auszeichnungen

Jeder zusammengefuehrte PR verdient Auszeichnungen basierend auf Meilensteinen. Schalten Sie Abzeichen fuer Ihren ersten PR, zehnten PR, Verdienstmeilensteine und Top-Ranglistenpositionen frei. Treten Sie auf der oeffentlichen Rangliste an, sortiert nach Gesamtverdienst. Alle Pull Requests sind oeffentlich.

Sichere Zahlungen

Kundenpraemienzahlungen werden ueber unterstuetzte Zahlungsanbieter abgewickelt. Wenn ein Kunde einen Pull Request genehmigt, wird automatisch abgerechnet und der Entwickler bezahlt.

Grundwerte

Transparenz

Jede Aufgabe, Praemie und Zahlung ist sichtbar. Keine versteckten Gebuehren, keine Ueberraschungen.

Fairness

Die Verguetung ist direkt an die gelieferte Arbeit geknuepft. Code wird nach seinen Qualitaeten bewertet, nicht danach, wer ihn geschrieben hat.

Open Source

Wir setzen uns fuer Open-Source-Prinzipien ein. Alle Pull Requests auf PrTask sind oeffentlich und foerdern den Wissensaustausch.

Gemeinschaftswirkung

PrTask betreibt eine florierende globale Gemeinschaft von Entwicklern und Kunden, die ueber Grenzen und Zeitzonen hinweg zusammenarbeiten.

Open-Source-Verpflichtung

PrTask basiert auf dem Open-Source-Oekosystem und dient ihm. Wir geben der Gemeinschaft zurueck, sponsern Open-Source-Maintainer und stellen sicher, dass jede Codezeile zugaenglich bleibt.
