Cara Kerja

Getting Started

1

Jelajahi Tugas Terbuka

Klien memposting tugas pengembangan dengan hadiah. Jelajahi berdasarkan kompleksitas, bahasa, atau jumlah hadiah.

2

Kirim Pull Request

Tulis kode Anda dan kirimkan pull request publik. Beberapa pengembang dapat mengirim PR untuk tugas yang sama.

3

Penilaian Kompleksitas

PR Anda otomatis dinilai berdasarkan file yang diubah, baris yang ditambahkan, kesulitan bahasa, dan cakupan pengujian.

4

Klien Meninjau dan Menyetujui

Klien meninjau semua PR yang dikirim dan memilih yang terbaik untuk disetujui. Saat disetujui, klien otomatis ditagih sejumlah hadiah.

5

Terima Pembayaran

Pengembang menerima pembayaran melalui metode pilihan mereka. Buka lencana pencapaian dan naiki papan peringkat penghasilan dengan setiap PR yang disetujui.

Untuk Klien

  1. Masuk dengan GitHub atau Google dan atur metode pembayaran Anda
  2. Buat tugas dengan deskripsi, tautan repositori, dan jumlah hadiah
  3. Pengembang mengirim pull request publik untuk tugas Anda
  4. Tinjau semua pull request yang dikirim dan setujui yang terbaik
  5. Anda secara otomatis ditagih hadiah dan pengembang dibayar
