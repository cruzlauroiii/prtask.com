Hubungkan akun Anda untuk mulai menghasilkan atau memposting tugas.

Untuk Pengembang

Jelajahi tugas, klaim pekerjaan, kirim PR, dan terima pembayaran.

Untuk Klien

Posting tugas dengan hadiah dan dapatkan kode berkualitas.

Mengapa PrTask?

PrTask menghubungkan pengembang dengan tugas berbayar open-source dan privat. Kirim kode, hasilkan uang, dan bangun portofolio Anda.

Hasilkan Uang

Dapatkan bayaran untuk setiap pull request yang digabung. Hadiah didanai di muka dan dirilis saat penggabungan.

Tanpa Biaya Platform

Pengembang menyimpan 100% hadiah. PrTask menagih klien, bukan kontributor.

PR Publik

Semua pekerjaan dikirim sebagai pull request nyata di GitHub. Kontribusi Anda terlihat dan dapat diverifikasi.

Cara Kerja

1

Masuk

Autentikasi dengan penyedia SSO pilihan Anda untuk membuat akun PrTask.

2

Jelajahi Tugas

Jelajahi tugas yang tersedia difilter berdasarkan bahasa, framework, atau jumlah hadiah.

3

Kirim PR

Klaim tugas, tulis kode, dan buka pull request terhadap repositori target.

4

Dapatkan Bayaran

Setelah PR Anda ditinjau dan digabung, hadiah otomatis dirilis ke akun Anda.

Keamanan dan Kepercayaan

PrTask hanya menggunakan SSO berbasis OAuth. Kami tidak pernah menyimpan kata sandi. Semua pembayaran disimpan dalam escrow hingga pekerjaan diverifikasi.

Butuh Bantuan?

Kunjungi dokumentasi kami untuk panduan, FAQ, dan sumber daya dukungan.
