Aplikasi Web

Gunakan PrTask langsung di browser Anda di prtask.com.

Petunjuk Instalasi

Android

Unduh file APK, buka di perangkat Anda, dan ikuti petunjuk di layar untuk menginstal.

Windows

Unduh penginstal, jalankan wizard pengaturan, dan ikuti petunjuknya.

iOS

Unduh dari App Store dan ketuk Instal. Aplikasi akan muncul di layar utama Anda.

macOS

Unduh file DMG, seret PrTask ke folder Aplikasi, dan luncurkan dari Launchpad.

Linux

Unduh paket untuk distribusi Anda dan instal menggunakan manajer paket Anda.

Verifikasi dan Keamanan

Semua unduhan PrTask ditandatangani secara digital dan diverifikasi. Periksa checksum SHA-256 setelah mengunduh.

Pemecahan Masalah

APK Android tidak terinstal. Apa yang harus saya lakukan?

Aktifkan 'Instal dari sumber tidak dikenal' di pengaturan perangkat Anda, lalu coba unduh lagi.

Aplikasi crash saat startup. Bagaimana cara memperbaikinya?

Pastikan perangkat Anda memenuhi persyaratan sistem minimum dan versi terbaru telah diinstal.
