Hoe het werkt

Getting Started

1

Bekijk open taken

Opdrachtgevers plaatsen ontwikkeltaken met beloningen. Blader op complexiteit, taal of beloningsbedrag.

2

Dien een Pull Request in

Schrijf uw code en dien een openbare pull request in. Meerdere ontwikkelaars kunnen PRs indienen voor dezelfde taak.

3

Complexiteitsscore

Uw PR wordt automatisch beoordeeld op basis van gewijzigde bestanden, toegevoegde regels, taalmoeilijkheid en testdekking.

4

Opdrachtgever beoordeelt en keurt goed

De opdrachtgever beoordeelt alle ingediende PRs en kiest de beste om goed te keuren. Bij goedkeuring wordt de opdrachtgever automatisch het beloningsbedrag in rekening gebracht.

5

Ontvang betaling

De ontwikkelaar ontvangt betaling via de gewenste methode. Ontgrendel prestatiebadges en klim in het verdienklassement met elke goedgekeurde PR.

Voor klanten

  1. Log in met GitHub of Google en stel je betaalmethode in
  2. Maak een taak aan met beschrijving, repository-link en beloningsbedrag
  3. Ontwikkelaars dienen openbare pull requests in voor je taak
  4. Beoordeel alle ingediende pull requests en keur de beste goed
  5. Je wordt automatisch belast en de ontwikkelaar wordt betaald
