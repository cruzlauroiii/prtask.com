Over PrTask

PrTask is een platform waar ontwikkelaars geld verdienen door pull requests in te dienen bij echte projecten.

Onze missie

PrTask bestaat om softwareontwikkeling toegankelijk, eerlijk en lonend te maken voor iedereen. Wij geloven dat getalenteerde ontwikkelaars overal ter wereld gelijke toegang verdienen tot zinvol werk en transparante beloning, ongeacht waar ze wonen.

Wat wij doen

Opdrachtgevers plaatsen ontwikkeltaken met beloningen. Ontwikkelaars claimen taken, schrijven code en dienen pull requests in. Zodra een PR is beoordeeld, goedgekeurd en samengevoegd, wordt de ontwikkelaar betaald op basis van de complexiteit van het werk.

Ons verhaal

PrTask werd opgericht met een simpel idee: wat als elke codebijdrage een duidelijke, gegarandeerde beloning had? Voortgekomen uit de frustratie van onbetaald open-source werk, creeerde PrTask een marktplaats waar pull requests de universele munteenheid van werk zijn.

Prestatiebadges

Elke samengevoegde PR levert prestatiebadges op basis van mijlpalen. Ontgrendel badges voor je eerste PR, tiende PR, verdiensten-mijlpalen en topposities op het scorebord. Concurreer op het publieke scorebord gerangschikt op totale verdiensten. Alle pull requests zijn openbaar.

Veilige betalingen

Klantbeloningsbetalingen worden verwerkt via ondersteunde betalingsproviders. Wanneer een klant een pull request goedkeurt, worden ze automatisch belast en wordt de ontwikkelaar betaald.

Kernwaarden

Transparantie

Elke taak, beloning en betaling is zichtbaar. Geen verborgen kosten, geen verrassingen.

Eerlijkheid

Compensatie is direct gekoppeld aan het geleverde werk. Code wordt beoordeeld op kwaliteit, niet op wie het schreef.

Open Source

We staan achter open source principes. Alle pull requests op PrTask zijn openbaar en bevorderen kennisdeling.

Community-impact

PrTask ondersteunt een bloeiende wereldwijde community van ontwikkelaars en klanten die samenwerken over grenzen en tijdzones heen.

Open Source toewijding

PrTask is gebouwd op en voor het open-source ecosysteem. We dragen bij aan de community, sponsoren open-source maintainers en zorgen ervoor dat elke regel code toegankelijk blijft.
