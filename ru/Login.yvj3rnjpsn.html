Подключите свой аккаунт, чтобы начать зарабатывать или публиковать задачи.

Войти через Google Войти через GitHub Войти через Microsoft Войти через Okta

Для разработчиков

Просматривайте задачи, берите работу, отправляйте PR и получайте оплату.

Для клиентов

Публикуйте задачи с вознаграждениями и получайте качественный код.

Почему PrTask?

PrTask связывает разработчиков с оплачиваемыми задачами open-source и частными проектами. Пишите код, зарабатывайте и создавайте портфолио.

Зарабатывайте деньги

Получайте оплату за каждый объединённый пул-реквест. Награды финансируются заранее и выплачиваются при слиянии.

Без комиссий платформы

Разработчики получают 100% награды. PrTask взимает плату с клиентов, никогда с разработчиков.

Публичные PR

Вся работа поставляется как настоящие пул-реквесты на GitHub. Ваши вклады видимы и проверяемы.

Как это работает

1

Войти

Аутентифицируйтесь через предпочитаемого SSO-провайдера для создания аккаунта PrTask.

2

Просмотреть задачи

Просматривайте доступные задачи с фильтрацией по языку, фреймворку или сумме награды.

3

Отправить PR

Возьмите задачу, напишите код и откройте пул-реквест в целевой репозиторий.

4

Получите оплату

После проверки и слияния вашего PR награда автоматически зачисляется на ваш счёт.

Безопасность и доверие

PrTask использует исключительно SSO на основе OAuth. Мы никогда не храним пароли. Все платежи хранятся на эскроу до верификации работы.
