PrTask Nedir - Pull Request Odul Pazari

PrTask, yazilim isine ihtiyac duyan musterileri pull request yoluyla gorevleri tamamlayan yetenekli gelistiricilerle bulusturan bir gelistirici pazaridir. Platform, musterilerin parasal odullerle gorevler yayinladigi ve gelistiricilerin incelemeden gecen yuksek kaliteli kod gondererek bu oduelleri kazandigi bir bounty tabanli modelle calisir. PrTask uzerindeki her gorev gercek bir GitHub deposuna baglidir.

Geleneksel freelancing platformlarindan farkli olarak PrTask yalnizca pull request tabanli calismaya odaklanir. Gelistiriciler acik gorevlere goz atabilir, cozumlerini pull request olarak gonderebilir ve kodlari onaylanip birlestirildiginde otomatik odeme alabilir.

PrTask gelistiricilerden platform ucreti almaz - kazanilan her odulun yuzde 100unu alirlar. Musteriler gorev olusturma sirasinda odul tutarini belirler ve odeme pull request onayindan sonra otomatik olarak islenir.
