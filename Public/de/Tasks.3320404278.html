Offene Aufgaben durchsuchen - Finden Sie bezahlte Entwicklungsarbeit

Die Aufgabenseite ist Ihr Einstiegspunkt fuer bezahlte Entwicklungsarbeit auf PrTask. Jede hier aufgefuehrte Aufgabe stellt echte Softwarearbeit dar, komplett mit einer Praemie, die Sie bei erfolgreicher Fertigstellung verdienen.

Jede Aufgabenkarte zeigt wesentliche Informationen auf einen Blick: die Komplexitaetsbewertung, den Praemienbetrag in USD, eine kurze Beschreibung und relevante Tags fuer die verwendeten Technologien.

Praemienbetraege werden von Kunden bei der Aufgabenerstellung festgelegt. PrTask erhebt keine Gebuehren fuer Entwickler - der angezeigte Betrag ist genau das, was Sie erhalten. Durchsuchen Sie die verfuegbaren Aufgaben und reichen Sie Ihre Loesung als Pull Request ein.
