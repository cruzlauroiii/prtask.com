Apa itu PrTask - Marketplace Pengembang untuk Bounty Pull Request

PrTask adalah marketplace pengembang yang menghubungkan klien yang membutuhkan pekerjaan perangkat lunak dengan pengembang terampil yang menyelesaikan tugas melalui pull request. Platform ini beroperasi dengan model berbasis bounty di mana klien memposting tugas dengan hadiah uang dan pengembang mendapatkan bounty tersebut dengan mengirimkan kode berkualitas tinggi yang lolos review. Setiap tugas di PrTask terkait dengan repositori GitHub nyata.

Berbeda dengan platform freelancing tradisional, PrTask berfokus secara eksklusif pada pekerjaan berbasis pull request. Pengembang dapat menelusuri tugas yang tersedia, mengirimkan solusi mereka sebagai pull request, dan menerima pembayaran otomatis setelah kode mereka disetujui dan digabungkan.

PrTask tidak membebankan biaya platform kepada pengembang - mereka menerima 100 persen dari setiap bounty yang diperoleh. Klien menetapkan jumlah bounty saat membuat tugas dan pembayaran diproses secara otomatis setelah persetujuan pull request.
