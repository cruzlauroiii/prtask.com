Dashboard PrTask Anda - Ikhtisar Platform dan Pelacakan Aktivitas

Dashboard menyediakan ikhtisar terpusat aktivitas platform di PrTask. Sekilas, Anda dapat melihat jumlah total tugas aktif yang tersedia, jumlah pengguna terdaftar, total pool bounty, dan jumlah pull request yang dikirim.

Bagi pengembang, Dashboard berfungsi sebagai titik awal untuk menilai lanskap pekerjaan yang tersedia saat ini. Bagi klien, Dashboard memberikan visibilitas status tugas yang telah dibuat, termasuk pull request yang menunggu review.

Dashboard diperbarui secara dinamis saat tugas baru dibuat, pull request dikirim, dan bounty dibayarkan. Dashboard akan terus berfungsi sebagai pusat komando Anda untuk melacak semua aspek aktivitas PrTask Anda.
