Wat is PrTask - De ontwikkelaarsmarktplaats voor Pull Request Bounties

PrTask is een ontwikkelaarsmarktplaats die klanten die softwarewerk nodig hebben verbindt met bekwame ontwikkelaars die taken voltooien via pull requests. Het platform werkt op een bounty-gebaseerd model waarbij klanten taken plaatsen met geldelijke beloningen en ontwikkelaars die bounties verdienen door hoogwaardige code in te dienen die de review doorstaat. Elke taak op PrTask is gekoppeld aan een echte GitHub-repository.

In tegenstelling tot traditionele freelance-platforms richt PrTask zich uitsluitend op pull request-gebaseerd werk. Ontwikkelaars kunnen openstaande taken bekijken, hun oplossingen indienen als pull requests en ontvangen automatische betaling zodra hun code is goedgekeurd en samengevoegd.

PrTask brengt geen platformkosten in rekening bij ontwikkelaars - ze ontvangen 100 procent van elke verdiende bounty. Klanten stellen het bounty-bedrag in bij het aanmaken van een taak en de betaling wordt automatisch verwerkt na goedkeuring van de pull request.
