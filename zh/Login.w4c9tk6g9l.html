面向开发者

面向客户

发布带赏金的任务，获得高质量代码交付。

为什么选择 PrTask？

PrTask 将开发者与付费的开源和私有任务连接起来。提交代码，赚取收入，建立你的作品集。

赚取收入

每个合并的拉取请求都能获得报酬。赏金预先资助，合并时释放。

无平台费用

开发者保留 100% 的赏金。PrTask 向客户收费，从不向贡献者收费。

公开 PR

所有工作都以 GitHub 上的真实拉取请求交付。您的贡献可见且可验证。

运作方式

1

登录

使用您首选的 SSO 提供商进行身份验证以创建 PrTask 账户。

2

浏览任务

按语言、框架或赏金金额筛选浏览可用任务。

3

提交 PR

认领任务，编写代码，并向目标仓库提交拉取请求。

4

获得报酬

您的 PR 被审查并合并后，赏金将自动释放到您的账户。

安全与信任

PrTask 专门使用基于 OAuth 的 SSO。我们从不存储密码。所有付款在工作验证前都保存在托管中。
