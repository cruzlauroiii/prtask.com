Ikonekta ang iyong account upang magsimulang kumita o mag-post ng mga gawain.

Mag-sign in gamit ang Google Mag-sign in gamit ang GitHub Mag-sign in gamit ang Microsoft Mag-sign in gamit ang Okta

Para sa mga Developer

Mag-browse ng mga gawain, kumuha ng trabaho, magsumite ng mga PR, at mabayaran.

Para sa mga Kliyente

Mag-post ng mga gawain na may bounty at makakuha ng de-kalidad na code.

Bakit PrTask?

Ikinokonekta ng PrTask ang mga developer sa mga bayad na open-source at pribadong gawain. Magpadala ng code, kumita ng pera, at buuin ang iyong portfolio.

Kumita ng Pera

Mabayaran para sa bawat na-merge na pull request. Ang mga bounty ay pinondohan nang maaga at inilalabas sa merge.

Walang Platform Fee

Natatanggap ng mga developer ang 100% ng bounty. Sinisingil ng PrTask ang mga kliyente, hindi kailanman ang mga contributor.

Mga Pampublikong PR

Lahat ng trabaho ay ipinapadala bilang mga tunay na pull request sa GitHub. Ang iyong mga kontribusyon ay nakikita at mave-verify.

Paano Gumagana

1

Mag-sign In

Mag-authenticate gamit ang iyong gustong SSO provider upang gawin ang iyong PrTask account.

2

Mag-browse ng mga Gawain

Mag-explore ng mga available na gawain na na-filter ayon sa wika, framework, o halaga ng bounty.

3

Magsumite ng mga PR

Kumuha ng gawain, isulat ang code, at magbukas ng pull request laban sa target na repositoryo.

4

Mabayaran

Kapag na-review at na-merge na ang iyong PR, awtomatikong inilalabas ang bounty sa iyong account.

Seguridad at Tiwala

Ang PrTask ay gumagamit lamang ng OAuth-based SSO. Hindi kami kailanman nag-iimbak ng mga password. Lahat ng mga bayad ay hawak sa escrow hanggang sa ma-verify ang trabaho.

Kailangan ng Tulong?

Bisitahin ang aming dokumentasyon para sa mga gabay, FAQ, at mga mapagkukunan ng suporta.
