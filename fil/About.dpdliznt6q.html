Tungkol sa PrTask

Ang PrTask ay isang platform kung saan kumikita ang mga developer sa pamamagitan ng pagsusumite ng mga pull request sa mga tunay na proyekto.

Ang Aming Misyon

Ang PrTask ay umiiral upang gawing accessible, patas, at kapaki-pakinabang ang software development para sa lahat. Naniniwala kami na ang mga mahuhusay na developer sa buong mundo ay nararapat na magkaroon ng pantay na access sa makabuluhang trabaho at transparent na kompensasyon kahit saan sila nakatira.

Ano ang Ginagawa Namin

Nagpo-post ang mga kliyente ng mga gawain sa development na may mga bounty. Kinukuha ng mga developer ang mga gawain, nagsusulat ng code, at nagsusumite ng mga pull request. Kapag na-review, na-approve, at na-merge ang isang PR, binabayaran ang developer batay sa pagiging kumplikado ng kanilang trabaho.

Ang Aming Kwento

Itinatag ang PrTask sa isang simpleng ideya: paano kung ang bawat kontribusyon ng code ay may malinaw at garantisadong bounty na nakakabit dito? Ipinanganak mula sa pagkabigo ng hindi binabayarang open-source na paggawa at mga hindi transparent na freelancing platform, inilunsad ang PrTask upang lumikha ng isang marketplace kung saan ang mga pull request ang unibersal na salapi ng trabaho. Mula sa aming unang bounty hanggang sa libu-libong nakumpletong gawain, lumago kami sa isang pandaigdigang komunidad ng mga developer at kliyente na nagbabahagi ng pangako sa transparent at merit-based na kolaborasyon.

Mga Achievement Badge

Bawat na-merge na PR ay nakakakuha ng mga achievement badge batay sa mga milestone. Mag-unlock ng mga badge para sa iyong unang PR, ikasampung PR, mga milestone sa kita, at mga nangungunang posisyon sa leaderboard. Makipagkompetensya sa pampublikong leaderboard na naka-rank ayon sa kabuuang kita. Lahat ng pull request ay pampubliko.

Mga Ligtas na Bayad

Ang mga bounty na bayad ng kliyente ay pinoproseso sa pamamagitan ng mga suportadong payment provider. Kapag na-approve ng kliyente ang isang pull request, awtomatiko silang sinisingil at mababayaran ang developer. Hindi kailangan ng advance na deposito. Pipiliin ng mga developer ang kanilang gustong paraan ng payout.

Mga Pangunahing Halaga

Transparency

Bawat gawain, bounty, at bayad ay nakikita. Eksaktong alam ng mga developer kung magkano ang kanilang kikitain bago magsimula, at eksaktong alam ng mga kliyente kung ano ang kanilang binabayaran. Walang mga nakatagong bayarin, walang mga sorpresa.

Pagiging Patas

Ang kompensasyon ay direktang nakatali sa trabahong naihatid. Rere-review ang code batay sa merito nito, hindi sa kung sino ang sumulat nito. Bawat developer ay nakikipagkompetensya sa pantay na larangan na may pantay na pagkakataon.

Open Source

Itinataguyod namin ang mga prinsipyo ng open-source. Lahat ng pull request sa PrTask ay pampubliko, nagtataguyod ng pagbabahagi ng kaalaman at mga pamantayan ng kalidad na pinapatnubayan ng komunidad sa bawat proyekto.

Epekto sa Komunidad

Pinapagana ng PrTask ang isang masigla at pandaigdigang komunidad ng mga developer at kliyente na nagtutulungan sa buong mundo upang sama-samang gumawa ng mas magandang software.

Pangako sa Open Source

Ang PrTask ay binuo sa at para sa open-source na ekosistema. Nag-aambag kami pabalik sa komunidad, nagsa-sponsor ng mga open-source maintainer, at tinitiyak na ang bawat linya ng code na isinumite sa aming platform ay nananatiling accessible. Ang aming proseso ng review ay pampubliko, ang aming leaderboard ay pampubliko, at ang aming pangako sa pagiging bukas ay hindi nagbabago.
