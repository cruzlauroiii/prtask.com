Mga Tagubilin sa Pag-install

Android

I-download ang APK file, buksan ito sa iyong device, at sundin ang mga on-screen prompt upang i-install.

Windows

I-download ang installer, patakbuhin ang setup wizard, at sundin ang mga prompt upang makumpleto ang installation.

iOS

I-download mula sa App Store at i-tap ang Install. Lalabas ang app sa iyong home screen.

macOS

I-download ang DMG file, i-drag ang PrTask sa iyong Applications folder, at ilunsad mula sa Launchpad.

Linux

I-download ang package para sa iyong distribution at i-install gamit ang iyong package manager.

Hindi mai-install ang Android APK. Ano ang dapat kong gawin?
