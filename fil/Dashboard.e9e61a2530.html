Ang Iyong PrTask Dashboard - Pangkalahatang Tanaw ng Platform at Pagsubaybay ng Aktibidad

Ang Dashboard ay nagbibigay ng sentralisadong pangkalahatang tanaw ng aktibidad ng platform sa PrTask. Sa isang tingin, makikita mo ang kabuuang bilang ng mga aktibong gawain, ang bilang ng mga rehistradong user, ang kabuuang bounty pool, at ang bilang ng mga na-submit na pull request.

Para sa mga developer, ang Dashboard ay nagsisilbing panimulang punto para suriin ang kasalukuyang landscape ng available na trabaho. Para sa mga kliyente, ang Dashboard ay nagbibigay ng visibility sa status ng mga nalikha na gawain, kabilang ang mga nakabinbing pull request na naghihintay ng review.

Ang Dashboard ay dynamic na nag-a-update habang nalilikha ang mga bagong gawain, nagsu-submit ng mga pull request, at binabayaran ang mga bounty. Ang Dashboard ay patuloy na magsisilbi bilang iyong command center para subaybayan ang lahat ng aspeto ng iyong PrTask na aktibidad.
