PrTask - La place de marche pour les primes de pull requests

PrTask est un marketplace de developpeurs qui connecte les clients ayant besoin de travail logiciel avec des developpeurs qualifies qui completent les taches via des pull requests. La plateforme fonctionne sur un modele base sur les primes. Chaque tache sur PrTask est liee a un vrai depot GitHub.

Contrairement aux plateformes de freelancing traditionnelles, PrTask se concentre exclusivement sur le travail base sur les pull requests. Les developpeurs peuvent parcourir les taches ouvertes, soumettre leurs solutions sous forme de pull requests et recevoir un paiement automatique.

PrTask ne facture aucun frais de plateforme aux developpeurs - ils recoivent 100 pour cent de chaque prime gagnee. Les clients fixent le montant de la prime lors de la creation de la tache et le paiement est traite automatiquement.
