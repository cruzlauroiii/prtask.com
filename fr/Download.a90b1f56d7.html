Download PrTask — Available on Multiple Platforms

PrTask is available as a native application on multiple platforms, giving you the flexibility to manage tasks, review pull requests, and track payments from your preferred device. The Android app provides a full-featured mobile experience with offline support, push notifications, and biometric authentication. The Windows desktop application offers a native experience with system tray integration and local file access. Linux users can download the standalone binary for x64 systems, and the web application is always accessible at prtask.com from any modern browser.

Each platform version of PrTask shares the same core functionality built with .NET MAUI and Blazor technology, ensuring a consistent experience across devices. Native apps offer additional capabilities like offline access to cached task data, push notifications for pull request reviews and payment updates, and biometric login for enhanced security. The web application requires no installation and works on any device with a modern browser, making it the fastest way to get started. iOS and macOS versions are currently in development and will be available in a future release.

Installation is straightforward on all supported platforms. Android users can download the APK directly and enable installation from unknown sources in their device settings. Windows users download the executable and run it — no installer required. Linux users download the binary, set the executable permission, and run it from the terminal. All downloads are verified and signed to ensure authenticity. Check the feature comparison table above to understand which capabilities are available on each platform, and refer to the troubleshooting section if you encounter any issues during installation.
