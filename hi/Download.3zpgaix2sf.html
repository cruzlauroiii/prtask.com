PrTask डाउनलोड करें

अपने डिवाइस पर PrTask ऐप प्राप्त करें।

Android

PrTask Android ऐप डाउनलोड करें - चलते-फिरते कार्य ब्राउज़ करें, PR सबमिट करें और कमाई ट्रैक करें।

सिस्टम आवश्यकताएँ: Android 8.0 या उच्चतर, 50 MB उपलब्ध संग्रहण

Android APK डाउनलोड करें

Windows

Windows 10 और बाद के संस्करण के लिए डेस्कटॉप ऐप।

सिस्टम आवश्यकताएँ: Windows 10 या उच्चतर, 64-बिट प्रोसेसर, 100 MB उपलब्ध संग्रहण

Windows EXE डाउनलोड करें

iOS

नेटिव iPhone और iPad ऐप।

सिस्टम आवश्यकताएँ: iOS 15 या उच्चतर, 50 MB उपलब्ध संग्रहण

macOS

Apple Silicon और Intel के लिए नेटिव Mac ऐप।

सिस्टम आवश्यकताएँ: macOS 12 (Monterey) या उच्चतर, 100 MB उपलब्ध संग्रहण

जल्द आ रहा है

Linux

Ubuntu, Fedora और अन्य डिस्ट्रीब्यूशन के लिए डेस्कटॉप ऐप।

सिस्टम आवश्यकताएँ: Ubuntu 20.04+ या Fedora 36+, 100 MB उपलब्ध संग्रहण

वेब ऐप

इंस्टॉलेशन निर्देश

Android

APK फ़ाइल डाउनलोड करें, इसे अपने डिवाइस पर खोलें और इंस्टॉल करने के लिए स्क्रीन पर दिए निर्देशों का पालन करें।

Windows

इंस्टॉलर डाउनलोड करें, सेटअप विज़ार्ड चलाएँ और निर्देशों का पालन करें।

iOS

App Store से डाउनलोड करें और इंस्टॉल पर टैप करें। ऐप आपकी होम स्क्रीन पर दिखाई देगा।

macOS

DMG फ़ाइल डाउनलोड करें, PrTask को एप्लिकेशन फ़ोल्डर में ड्रैग करें और Launchpad से लॉन्च करें।

Linux

अपने डिस्ट्रीब्यूशन के लिए पैकेज डाउनलोड करें और अपने पैकेज मैनेजर से इंस्टॉल करें।

सत्यापन और सुरक्षा

सभी PrTask डाउनलोड डिजिटल रूप से हस्ताक्षरित और सत्यापित हैं। डाउनलोड के बाद SHA-256 चेकसम जाँचें।

समस्या निवारण

Android APK इंस्टॉल नहीं होता। मुझे क्या करना चाहिए?

अपने डिवाइस सेटिंग्स में 'अज्ञात स्रोतों से इंस्टॉल' सक्षम करें, फिर डाउनलोड पुनः प्रयास करें।

ऐप स्टार्टअप पर क्रैश होता है। मैं इसे कैसे ठीक करूँ?

सुनिश्चित करें कि आपका डिवाइस न्यूनतम सिस्टम आवश्यकताओं को पूरा करता है और नवीनतम संस्करण इंस्टॉल है।
