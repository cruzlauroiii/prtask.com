كيف يعمل

Getting Started

1

تصفح المهام المفتوحة

ينشر العملاء مهام تطوير مع مكافآت. تصفح حسب التعقيد أو اللغة أو مبلغ المكافأة.

2

قدم طلب سحب

اكتب الكود الخاص بك وقدم طلب سحب عام. يمكن لعدة مطورين تقديم PRs لنفس المهمة.

3

تقييم التعقيد

يتم تقييم PR الخاص بك تلقائيًا بناءً على الملفات المعدلة والأسطر المضافة وصعوبة اللغة وتغطية الاختبارات.

4

العميل يراجع ويوافق

يراجع العميل جميع طلبات السحب المقدمة ويختار الأفضل للموافقة عليه. عند الموافقة، يتم تحصيل مبلغ المكافأة من العميل تلقائيًا.

5

احصل على الدفع

يتلقى المطور الدفع عبر طريقته المفضلة. افتح شارات الإنجاز وتسلق لوحة متصدري الأرباح مع كل PR معتمد.

للعملاء

  1. سجّل الدخول باستخدام GitHub أو Google وأعدّ طريقة الدفع
  2. أنشئ مهمة بوصف ورابط المستودع ومبلغ المكافأة
  3. يقدم المطورون طلبات سحب عامة لمهمتك
  4. راجع جميع طلبات السحب المقدمة ووافق على الأفضل
  5. يتم تحصيل المكافأة تلقائياً ويُدفع للمطور
