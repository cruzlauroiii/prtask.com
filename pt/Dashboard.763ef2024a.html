Seu Painel PrTask - Visao geral da plataforma e acompanhamento de atividades

O Painel fornece uma visao centralizada da atividade da plataforma no PrTask. Em um relance, voce pode ver o numero total de tarefas ativas disponiveis, o numero de usuarios registrados, o pool total de recompensas e a contagem de pull requests enviados.

Para os desenvolvedores, o Painel serve como ponto de partida para avaliar o cenario atual de trabalho disponivel. Para os clientes, o Painel fornece visibilidade do status das tarefas criadas, incluindo pull requests pendentes aguardando revisao.

O Painel se atualiza dinamicamente conforme novas tarefas sao criadas, pull requests sao enviados e recompensas sao pagas. O Painel continuara servindo como seu centro de comando para rastrear todos os aspectos de sua atividade no PrTask.
