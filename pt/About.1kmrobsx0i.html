Sobre o PrTask

PrTask é uma plataforma onde desenvolvedores ganham dinheiro enviando pull requests para projetos reais.

Nossa missão

PrTask existe para tornar o desenvolvimento de software acessível, justo e gratificante para todos. Acreditamos que desenvolvedores talentosos em qualquer lugar merecem acesso igualitário a trabalhos significativos e remuneração transparente, independentemente de onde moram.

O que fazemos

Clientes publicam tarefas de desenvolvimento com recompensas. Desenvolvedores reivindicam tarefas, escrevem código e enviam pull requests. Quando uma PR é revisada, aprovada e mesclada, o desenvolvedor é pago com base na complexidade do trabalho.

Nossa historia

PrTask foi fundado com uma ideia simples: e se cada contribuicao de codigo tivesse uma recompensa clara e garantida? Nascido da frustracao do trabalho open-source nao remunerado, PrTask criou um mercado onde pull requests sao a moeda universal do trabalho.

Insignias de conquista

Cada PR mesclado ganha insignias de conquista baseadas em marcos. Desbloqueie insignias pelo seu primeiro PR, decimo PR, marcos de ganhos e posicoes no ranking. Compita no ranking publico ordenado por ganhos totais. Todos os pull requests sao publicos.

Pagamentos seguros

Os pagamentos de recompensas sao processados atraves de provedores de pagamento suportados. Quando um cliente aprova um pull request, e cobrado automaticamente e o desenvolvedor e pago.

Valores fundamentais

Transparencia

Cada tarefa, recompensa e pagamento e visivel. Sem taxas ocultas, sem surpresas.

Equidade

A remuneracao esta diretamente ligada ao trabalho entregue. O codigo e avaliado por seus meritos, nao por quem o escreveu.

Open Source

Defendemos os principios open source. Todos os pull requests no PrTask sao publicos, promovendo o compartilhamento de conhecimento.

Impacto na comunidade

PrTask impulsiona uma comunidade global de desenvolvedores e clientes que colaboram alem de fronteiras e fusos horarios.

12.500+ Tarefas concluidas
3.200+ Desenvolvedores ativos
85+ Paises representados
US$2,4M+ Pago a desenvolvedores

Compromisso open source

PrTask e construido sobre e para o ecossistema open-source. Contribuimos para a comunidade, patrocinamos mantenedores e garantimos que cada linha de codigo permaneca acessivel.
