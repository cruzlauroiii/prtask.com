Como funciona

Getting Started

1

Explorar tarefas abertas

Clientes publicam tarefas de desenvolvimento com recompensas. Navegue por complexidade, linguagem ou valor da recompensa.

2

Enviar uma Pull Request

Escreva seu código e envie uma pull request pública. Vários desenvolvedores podem enviar PRs para a mesma tarefa.

3

Pontuação de complexidade

Sua PR é pontuada automaticamente com base nos arquivos alterados, linhas adicionadas, dificuldade da linguagem e cobertura de testes.

4

O cliente revisa e aprova

O cliente revisa todas as PRs enviadas e escolhe a melhor para aprovar. Na aprovação, o cliente é cobrado automaticamente pelo valor da recompensa.

5

Receba o pagamento

O desenvolvedor recebe o pagamento pelo método preferido. Desbloqueie conquistas e suba no ranking de ganhos a cada PR aprovada.

Para clientes

  1. Faca login com GitHub ou Google e configure seu metodo de pagamento
  2. Crie uma tarefa com descricao, link do repositorio e valor da recompensa
  3. Desenvolvedores enviam pull requests publicos para sua tarefa
  4. Revise todos os pull requests enviados e aprove o melhor
  5. Voce e cobrado automaticamente e o desenvolvedor recebe o pagamento
